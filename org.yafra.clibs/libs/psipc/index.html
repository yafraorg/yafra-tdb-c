Yafra.org REPOSITORY: IPC

Kurzbeschreibung

Folgende Funktionen sind beinhaltet:
  • PSIPCserver()
  • PSIPCclient()
  • PSIPCread()
  • PSIPCpipeInOut()
Der source code befindet sich in $PSIPCLIB und die library kann mit make all erstellt werden. Die noetigen manuals koennen mit gmake man install.man generiert und in $PISOMAN installiert werden.

Diese library enthaelt server und client Funktionen fuer eine interprocess communication ueber pipe oder socket.

Specials

Folgende Variablen muessen im main() als globale Variable definiert und gesetzt werden: 
int DebugFlag;
void (*PSclient)(int, char**, int);

char *PSIPCservicetitle; /* name of service for displaying */

char *PSIPCservicename;  /* /etc/services name */

char *PSIPCprot;
Für Windows kann die library als "plain win32" oder mit "nutcracker" erstellt werden. Im Makefile ist das define "ps_winplain" zu definieren, falls eine reine win32 library noetig ist.

Fuer Windows NT/95 ist die library wsock32.lib und advapi32.lib zu linken ! Mit Nutcracker Support ebenfalls nutc.lib und socket.lib.

Offene Punkte

  • Im Daemon/Service Modul SYSLOG/EVENT noch einfuegen.
