action groups currently defined:


SP Standardbuchung
==================

  1 Status Eingabe - Option        : Eingabe            -> Option          :  Datei, Aktion automatisch sofort
  2 Invoice from state option      : Definitiv          -> Rechnung offen  :  Datei, Aktion automatisch sofort
  3 SP Voucher                     : Rechnung bezahlt   -> Erledigt        :  Datei, Aktion automatisch sofort

TO Requestbuchung
=================

  1 Anfrage an Leistungsanbieter   : Eingabe            -> Anfrage          :  Datei, Aktion automatisch sofort
  2 TO Invoice from state req OK   : Anfrage bestaetigt -> Rechnung offen   :  Datei, Aktion automatisch sofort
  3 TO Voucher                     : Rechnung bezahlt   -> Erledigt         :  Datei, Aktion automatisch sofort
